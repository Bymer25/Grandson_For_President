# План развития GrandsonSound (Roadmap)

## 1. Краткосрочные улучшения (3–6 месяцев)
- Реальные аудиофайлы: заменить тестовые WAV-ссылки на собственные MP3 (лицензированные), разместить на CDN.
- Бэкенд и база данных: разработать серверную часть (Node.js/Python) с PostgreSQL для хранения пользователей, подписок, плейлистов.
- Регистрация и вход: подключить реальную аутентификацию (JWT, OAuth VK/Google).
- Платёжная интеграция: подключить агрегатор (CloudPayments / ЮKassa) для приёма платежей.
- Личный кабинет: добавить страницу профиля с историей платежей, управлением подпиской.
- Мобильная адаптация: доработать интерфейс для комфортного использования на телефонах.

## 2. Среднесрочные улучшения (6–12 месяцев)
- Социальные функции:
  - Совместные сессии (синхронное прослушивание с друзьями).
  - Общие плейлисты с возможностью редактирования несколькими пользователями.
  - Чат и комментарии к трекам.
- Персонализация:
  - Рекомендательная система на основе истории прослушиваний.
  - «Умные» плейлисты (например, «Повтор»).
- Мобильное приложение: React Native / Flutter для iOS и Android.

## 3. Долгосрочные улучшения (1–2 года)
- Эксклюзивный контент: интервью, акустические версии, ранние релизы.
- Интеграция с умными колонками (Яндекс.Станция, Apple HomePod).
- Аудио высокого разрешения (Hi-Res) для премиум-подписчиков.
- Партнёрства с лейблами для расширения каталога.
- Геймификация: достижения, челленджи среди пользователей.
