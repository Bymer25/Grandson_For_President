# GrandsonSound — музыкальный стриминг с фокусом на Chiraq Drill

GrandsonSound — это веб-приложение для прослушивания музыки, созданное как учебный проект и прототип музыкального стримингового сервиса. Основная специализация — жанр Chiraq Drill и такие исполнители, как King Von, NLE Choppa, Chief Keef, Lil Durk. Проект включает в себя каталог треков, плеер с реальным воспроизведением, систему тарифов (подписок), страницы авторизации и регистрации, а также сравнительную таблицу с конкурентами.

---

## 🎯 Цели и аудитория

- Цель: продемонстрировать работу фронтенд-части музыкального сервиса, реализовать адаптивный дизайн, плеер с поддержкой плейлистов и адаптивного битрейта, а также продумать пользовательские сценарии (выбор тарифа, вход, регистрация).
- Целевая аудитория: поклонники жанра Drill, молодые люди 16–35 лет, которые ищут удобный способ слушать любимых исполнителей и открывать новую музыку в этом жанре.

---

## ✨ Основные функции

- Каталог треков исполнителей King Von, NLE Choppa, Chief Keef, Lil Durk.
- Полноценный аудиоплеер с отображением обложек, управлением воспроизведением и автоматическим переходом к следующему треку.
- Страница тарифов с несколькими вариантами подписки (Студенческий, Базовый, Премиум, Семейный).
- Формы входа и регистрации (имитация отправки данных).
- Сравнительная таблица с конкурентами (Яндекс.Музыка, VK Музыка, Apple Music).
- Адаптивный дизайн для мобильных устройств.
- Навигация между страницами с сохранением активного пункта меню.

---

## 🛠 Технологический стек

- Frontend: HTML5, CSS3 (кастомные переменные, flex/grid), JavaScript (ES6+).
- Библиотеки и иконки: Font Awesome 6, Google Fonts (Inter).
- Аудиоплеер: стандартный HTML5 <audio> с кастомным управлением.
- Инструменты разработки: любой текстовый редактор (VS Code), для запуска достаточно браузера.

## 🚀 Запуск проекта (локально)

Проект полностью статический, для запуска не требуется серверная часть. Достаточно открыть любой HTML-файл в современном браузере.

### Способ 1: открыть файл напрямую

1. Скачайте или склонируйте репозиторий.
2. Перейдите в папку grandsonsound.
3. Дважды кликните на файл index.html — он откроется в браузере, и вы сможете перемещаться по страницам через навигационное меню.

### Способ 2: использовать локальный сервер (рекомендуется для разработки)

Если вы используете VS Code, установите расширение Live Server и запустите index.html через него. Это позволит автоматически обновлять страницу при изменениях в коде и избежать возможных проблем с CORS при загрузке аудиофайлов (хотя в нашем случае аудио берётся с внешних ресурсов).

После запуска откройте в браузере http://localhost:5500 (порт может отличаться).

## 🔧 Примечания для разработки

- Все изображения (обложки треков) берутся из открытых источников (Wikimedia). При необходимости их можно заменить на локальные файлы в папке assets/images/.
- Аудиофайлы в демо-версии используют тестовые WAV-файлы с университетского сервера. Для продакшена их нужно заменить на собственные или лицензированные треки, размещённые на CDN.
- Для реализации реальной оплаты потребуется интеграция с платёжным агрегатором (CloudPayments, ЮKassa и др.), а также бэкенд для обработки подписок.
